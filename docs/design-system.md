# Design System — A9 Global Travels & Tours

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Gold 400 | `#D4AF37` | Primary accent, CTAs, luxury highlights |
| Sky 600 | `#1A7FDB` | Secondary, links, sky gradients |
| Sky 900 | `#0A3D6B` | Deep backgrounds, hero overlays |
| Cloud White | `#FAFCFF` | Page backgrounds |
| Neon Cyan | `#00F5D4` | Neon tree glow accents |
| Neon Purple | `#9B5DE5` | Futuristic highlights |

## Typography

- **Headings:** Playfair Display (serif, luxury feel)
- **Body:** Inter (clean, readable)

## Visual Style

- Parallax hero with lake + futuristic skyline
- Glassmorphism cards with gold borders
- Subtle neon glow on hover states
- Cloud-like soft gradients (blue → white)
